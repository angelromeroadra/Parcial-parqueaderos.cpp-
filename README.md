INNOVACIONES RESUMEN:  

Cada ingreso y salida queda guardado permanentemente en un archivo de texto en el disco. Al reiniciar el programa, el historial no se pierde.   
  
El mapa se dibuja en consola con colores diferentes para cada tipo de celda: verde para libre, rojo para ocupado, amarillo para entrada, etc.  
  
Al registrar un ingreso, el sistema dibuja el camino exacto con * desde la entrada hasta el espacio asignado.  
  
La opción de estadísticas muestra una barra tipo [====----] que indica visualmente qué tan lleno está el parqueadero, y lanza una alerta si supera el 80%  
  
Al buscar una placa activa, el sistema muestra cuánto debería pagar ese vehículo en ese momento exacto, sin necesidad de sacarlo.  
  
El sistema solo acepta placas con formato colombiano válido: exactamente 3 letras seguidas de 3 números (ej: ABC123). Cualquier otro formato es rechazado.
